# Put your probability and statistics PDF books in this directory

This directory should contain the PDF files of probability and statistics books that need to be scanned for errors.

## Supported file types
- PDF files (*.pdf)

## Example structure
```
books/
  ├── probability_theory.pdf
  ├── statistics_fundamentals.pdf
  └── advanced_stats.pdf
```

The scanner will process all PDF files in this directory when using the `--directory` option.
